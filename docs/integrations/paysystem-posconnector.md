# Требования
Используемая версия Айтида и Айтида РМК должна быть версии 4.7.3 и выше.
Требуется загрузить из центра загрузок [актуальные профили](https://itida.ru/obnovleniya-i-profili/profili-oborudovaniya-algoritmy-parametry-i-funktsii) работы с оборудованием.

# Установка
Для интеграции необходимо установить драйвер PosConnector 32х разрядной версии.

> Работа с драйвером x64 не поддерживается
{.is-warning}

Инструкция по установке драйвера и настройке связи с пинпадом доступна по ссылке: http://service.pbfgroup.ru/faqpbf/POS/podklyuchenie_k_windows_2.htm

# Настройка ПО
1. В Айтиде необходимо создать карточку оборудования по следующему шаблону:

![eq_posconnector.png](/images/integrations/posconnector/eq_posconnector.png){.align-center}
**Наименование**: указать наименование для оборудования
**Тип оборудования**: выбрать "Платежная система"
**Профиль**: выбрать профиль с названием "Платежная система через драйвер POSConnector"
**Краткое имя**: указать краткое наименование для платежной системы, которое будет отображаться в интерфейсе кассира.
**Склад нахождения и Денежный карман**: выбрать требуемые значения
**Вид оплаты**: выбрать "Безналичные"
**Копий слипа**: указать количество печатаемых копий слипа.

2. Сохранить карточку оборудования и добавить её в список оборудования для рабочего места в схеме РМК.

![scheme_eq.png](/images/integrations/posconnector/scheme_eq.png){.align-center}

3. После этого запустить РМК на рабочем месте и авторизоваться под пользователем с правами настроек.
	Перейти в параметры программы. Это можно сделать либо через отдельную клавиатурную функцию, либо через экранные кнопки, если включено отображение правой панели.

![service.png](/images/integrations/qrbyyandexpay/service.png =500x300){.align-center}
![paramsbutton.png](/images/integrations/qrbyyandexpay/paramsbutton.png =500x300){.align-center}

4. Откроется окно настроек кассового приложения.
	На вкладке **Основные параметры** нажмите кнопку **"Настройка оборудования"**.

![btnequipment.png](/images/integrations/qrbyyandexpay/btnequipment.png =700x520){.align-center}

5. Выбрать из списка оборудование для настроенной платежной системы и нажать на кнопку с тремя точками.

![eq_rmk.png](/images/integrations/posconnector/eq_rmk.png){.align-center}

6. Будет открыто окно настройки параметров платежной системы, в котором необходимо указать ИД терминала и параметры связи с платежной системой.

> Набор настраиваемых параметров зависит от используемой версии Айтида
{.is-info}

![dll_params1.png](/images/integrations/posconnector/dll_params1.png =600x200){.align-center}
![dll_params2.png](/images/integrations/posconnector/dll_params2.png =600x200){.align-center}

7. Для сохранения параметров необходимо нажать кнопку "**Продолжить**" в окнах ввода параметров платежной системы и в списке оборудования, а затем нажать кнопку "**Применить**" в параметрах приложения.

Настройка завершена. Протестируйте оплату заказа, чтобы проверить работу сервиса.
  
# Оплата
1. Для оплаты чека через подключенный банковский терминал необходимо в окне оплаты указать сумму оплаты по виду "Безнал".

![bankpay.png](/images/integrations/paysystem-arcus2/bankpay.png){.align-center}

При наличии нескольких подключенных терминалов требуется вывести окно распределения и указать сумму оплаты для требуемой платежной системы.

![bank2_pay.png](/images/integrations/posconnector/bank2_pay.png =500x500){.align-center}

2. Нажмите кнопку Продолжить → Печать чека, чтобы вызвать операции оплаты и закрытия чека.
	Операция оплаты будет выведена на подключенном банковском терминале и будет запущен процесс ожидания платежа.
3. После того как клиент произведет оплату на банковском пинпаде и оплата пройдет успешно чек будет завершен, текущая регистрация будет очищена.
