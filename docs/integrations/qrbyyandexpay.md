# Требования
Используемая версия Айтида и Айтида РМК должна быть версии 4.7.3 и выше.
Требуется загрузить из центра загрузок [актуальные профили](https://itida.ru/obnovleniya-i-profili/profili-oborudovaniya-algoritmy-parametry-i-funktsii) работы с оборудованием.

# Настройка ПО
1. В Айтиде необходимо создать карточку оборудования по следующему шаблону:

![equipment.png](/images/integrations/qrbyyandexpay/equipment.png){.align-center}
**Наименование:** указать наименование для оборудования
**Тип оборудования:** выбрать "Платежная система"
**Профиль:** выбрать профиль с названием "QR-код от Яндекс Пэй"
**Краткое имя:** указать краткое наименование для платежной системы, которое будет отображаться в интерфейсе кассира.
**Склад нахождения** и **Денежный карман**: выбрать требуемые значения
**Вид оплаты:** выбрать "Безналичные"

2. Сохранить карточку оборудования и добавить её в список оборудования для рабочего места в схеме РМК.

![rmk_scheme.png](/images/integrations/qrbyyandexpay/rmk_scheme.png){.align-center}

3. После этого запустить РМК на рабочем месте и авторизоваться под пользователем с правами настроек.
	Перейти в параметры программы. Это можно сделать либо через отдельную клавиатурную функцию, либо через экранные кнопки, если включено отображение правой панели.

![service.png](/images/integrations/qrbyyandexpay/service.png =500x300){.align-center}
![paramsbutton.png](/images/integrations/qrbyyandexpay/paramsbutton.png =500x300){.align-center}

4. Откроется окно настроек кассового приложения.
	На вкладке **Основные параметры** нажмите кнопку **"Настройка оборудования"**.

![btnequipment.png](/images/integrations/qrbyyandexpay/btnequipment.png =700x520){.align-center}

5. Выбрать из списка оборудование "QR-код от Яндекс Пэй" и нажать на кнопку с тремя точками.

![eqlist.png](/images/integrations/qrbyyandexpay/eqlist.png){.align-center}

6. В параметрах настройки платежной системы **QR-код от Яндекс Пэй** указать следующие параметры:
- **Токен пользователя** - [токен](/https://pay.yandex.ru/docs/ru/qr-code/#token), который вы выпустили в личном кабинете Яндекс Пэй.
- **QR табличка** — номер QR-таблички, которую вы получили. Выбор таблички доступен после ввода токена
- **Время жизни QR кода (секунд)** - время, через которое QR-код будет деактивирован.
- **Дополнительный вывод QR** - определяет где нужно вывести QR дополнительно, кроме QR таблички. Доступен дополнительный вывод QR кода на экране покупателя, если подключено приложение экрана покупателя.

![qrparams.png](/images/integrations/qrbyyandexpay/qrparams.png){.align-center}

7. По кнопке "**Проверить токен**" можно протестировать корректность указания токена пользователя.

![qrcheck.png](/images/integrations/qrbyyandexpay/qrcheck.png){.align-center}

8. Для сохранения параметров необходимо нажать кнопку "**Продолжить**" в окнах ввода параметров платежной системы и в списке оборудования, а затем нажать кнопку "**Применить**" в параметрах приложения.

Настройка завершена. Протестируйте оплату заказа, чтобы проверить работу сервиса.

# Оплата
1. Для оплаты чека через подключенный сервис **"QR-код от Яндекс Пэй"** необходимо вызвать окно распределения оплат по платежным системам и указать сумму оплаты.

![cheque_pay.png](/images/integrations/qrbyyandexpay/cheque_pay.png =500x500){.align-center}

2. Нажмите кнопку Продолжить → Печать, чтобы вызвать операции оплаты и закрытия чека.
	QR-код будет выведен на QR-табличке и дополнительно может быть выведен на экране покупателя, в зависимости от настроек.
	Тут же запустится процесс ожидания оплаты по QR.
3. Попросите клиента отсканировать табличку с QR-кодом, которую вам доставили, и оплатить заказ. Чек будет завершен, текущая регистрация будет очищена.

	Чтобы прервать процесс ожидания оплаты, нажмите кнопку **Нет** в диалоге запроса повторной проверки оплаты или клавишу <kbd>Esc</kbd> на клавиатуре.

	Если ожидание оплаты отменили, будет вызвана функция отмены оплаты (деактивации QR).
